2. Third Person Shooter

Developed a Third-person shooter game using Unreal 5 and C++. Implemented movement for the player and the enemies along with the shooting mechanics. Implemented animation blueprints for both the player and the enemy. Blend shapes were used for smoother transitioning between different directional movements. Used behavior trees and blackboards to set up AI mechanics for the enemies.
